# git

## Pretty log

```sh
git log --graph --oneline --all --decorate --date-order
```

## Interactive patch selection

```sh
git commit -p -v
```

## Lines of code written today

```sh
git diff --shortstat "@{0 day ago}"
```

## Ignore file mode

```sh
git config core.fileMode false
```
